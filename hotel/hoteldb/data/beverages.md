# Beverages List

## Parameters
- id
- name
- url
- description
- price
- state : [ok, planned, unavailable, endoflife, error] (default=ok)
- calories      (default=0)
- vegetarian    (default=false)
- vegan         (default=false)
- allergens     (default=none)
- alcoholic     (default=false)
